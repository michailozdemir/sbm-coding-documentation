CSS Методология

Данный раздел покрывает вопрос CSS методологии, какую методологию использовать, как правильно называть классы, как НЕ стоит стилизировать элементы.

БЭМ

БЭМ (Блок-Элемент-Модификатор) является одной из самых популярных и удобных методологий для использования. Изначально может показаться что она сложная, но это не так. Ее использование со временем упростит верстку, именование классов, редактирование уже готовых страниц, проектов и так далее.

Обозначения:

-- - (двойной дефис) означает вариант элемента.
__ - (двойное подчёркивание) означает дочерний элемент.

Пример: 

            <button class='btn btn--warning'> <!-- Variant .btn — .btn--warning -->
  <div class='btn__text'></div> <!-- Child element .btn — .btn__text-->
</button>
        
            .btn--warning {
/* Here you can understand that the style will be applied to warning buttons */
}

.btn__text {
/* Same here. It is button text! */
}

Пример (элемент списка): 

            <li class="list__item item">
  <h3 class="item__title">Title</h3>
  <p class="item__text">Description</p>
</li>

Префикс js

Если элемент используется для JavaScript, не стоит полагаться на CSS-стили. Проще задать специальный префикс вроде js

Пример: 

            <a class="btn btn--main js-demo">Click for demo!</a>

Расположения текста

Для расположения текста по определенной заданой стороне можно задать отдельным классом, который легко использовать и запомнить text-align.
И его вариантами: --left, --center, --right

Пример: 

            <p class="item__text text-align--center">Item text!</p>
            
        
            .text-align {
  &--left {
    text-align: left;
  }

  &--center {
    text-align: center;
  }

  &--right {
    text-align: right;
  }
 }

Префикс is

Манипуляция состоянием происходит очень часто. Поэтому придерживаться строгого соглашения наименования для состояния будет очень полезно.
Пример ниже объясняет то, что пункт меню является активным. Стилизирование активного пункта выполняется сменой цвета.

Пример: 

            <a class="menu__link is-active">Главная</li>
        
            .is-active {
  color: $primary-color;
 }

Слова, которые стоит использовать в CSS-классах

Унифицирование слов, используемых в классах поможет в работе каждому кодеру в команде.
Стоит вынести основные классы для разных элементов на странице.

Изображения

image, picture - картинка

icon - иконка

logo, picture - логотип

avatar, picture - аватар, профиль пользователя


Текст

title - заголовок

subtitle - подзаголовок

heading - заголовок (меньший по важности)

text - текст

link - ссылка

copyright - копирайт

description - описание


Блоки

header - шапка

footer - подвал, нижняя часть секции или блока

section - секция, раздел на странице

sidebar - боковая колонка (страницы или элемента)


Списки

list - список

item - айтем, карточка списка, элемент списка


Раскладка

wrapper - обертка, подойдет для кнопок или чего-то подобного


Элементы управления

btn - кнопка

dropdown - выпадающий список


Размеры

big - большой

medium - средний

small - маленький
